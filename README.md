# MD2Card

Markdown 转知识卡片 / 小红书风格图片的本地工具。

## 当前状态

这个项目目前已暂停维护，保留为公开作品集和阶段性内容工具记录。

它曾经用于把 Markdown 文本快速转换成适合分享的图文卡片。后来随着 GPT-4o / GPT Image 等图像生成和排版能力快速增强，这类独立 Markdown 转图工具的必要性明显下降。

## 它想解决什么问题

当时的核心问题是：内容写在 Markdown 里很方便，但发到小红书、朋友圈或其他视觉平台时，需要变成更好看的图片卡片。

这个项目想解决：

- Markdown 文本快速排版成卡片
- 支持标题、列表、引用、代码等基础语法
- 实时预览视觉效果
- 一键导出 PNG
- 长文自动分页
- 多主题、多样式切换
- 不依赖后端，打开本地页面即可使用

它本质上是一个轻量的“内容再包装工具”。

## 设计思路

设计上追求的是低门槛和即时反馈：

- 左侧编辑 Markdown
- 右侧实时预览卡片
- 用户通过主题、字号、宽度等参数调节视觉风格
- 使用前端能力将页面渲染为图片
- 长内容自动拆分成多张图，适合社媒发布

这个项目不是为了做复杂设计软件，而是让“文本 -> 可发布图片”尽量快。

## 后来的反思

这个方向在当时很实用，但现在已经被更强的 AI 图像和排版能力部分替代。

当模型可以直接理解内容、生成视觉稿、控制版式，并且快速迭代风格时，手动维护一套固定模板系统的价值就降低了。

更重要的是，内容的核心竞争力并不在于 Markdown 到图片的转换，而在于选题、观点、表达、结构和业务理解。视觉工具只是最后一公里。

## 是否继续维护

暂不维护。

后续如果重新启动，可能不会继续做传统模板编辑器，而会转向：

- 内容结构分析
- 多平台发布格式适配
- AI 生成视觉版本
- 结合品牌风格的自动化排版

## 技术栈

项目主要基于 HTML、CSS 和 JavaScript 构建，使用前端 Markdown 解析和截图导出能力完成本地卡片生成。

## 作品集说明

这个仓库展示的是一次内容工具产品化尝试。它现在不是主力工具，但记录了从文本内容到社媒视觉表达的一段探索。
