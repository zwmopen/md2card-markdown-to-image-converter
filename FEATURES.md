# MD2Card 高级功能指南

## 🚀 全新升级功能

基于对优秀开源项目的研究和融合，MD2Card现已升级为功能强大的卡片制作工具，集成了多个开源项目的优点。

## ✨ 核心功能升级

### 1. 高级编辑器 (Advanced Editor)
**灵感来源**: mdx-editor, Monaco Editor
- 🛠️ **富文本工具栏** - 一键格式化文本
- ⌨️ **智能快捷键** - Ctrl+B粗体，Ctrl+I斜体，Tab缩进
- 📁 **拖拽上传** - 直接拖拽图片到编辑器
- 😊 **表情符号** - 内置emoji选择器
- 🔍 **全屏编辑** - 专注写作模式
- 📝 **行号显示** - 可选的代码编辑体验
- 💾 **自动保存** - 本地存储防丢失

### 2. 智能模板系统 (Template System)
**灵感来源**: Notion模板, 各类CMS系统
- 📋 **预设模板**:
  - 🌸 小红书生活分享
  - 💄 小红书美妆教程  
  - 🍰 小红书美食分享
  - 📚 学习笔记模板
  - 💼 工作总结模板
- 🎨 **分类筛选** - 按类别快速查找模板
- 📝 **表单填写** - 结构化内容输入
- 🔧 **自定义模板** - 创建个人专属模板
- 📤 **模板导入导出** - 分享和备份模板

### 3. 动画效果系统 (Animation Effects)
**灵感来源**: Framer Motion, AOS动画库
- 🎬 **页面动画** - 淡入、滑入、缩放等8种动画
- 🔄 **卡片切换** - 流畅的页面切换效果
- ✨ **交互反馈** - 按钮点击、悬停效果
- 🎉 **成功动画** - 操作成功的视觉反馈
- 📊 **进度动画** - 数字计数、进度条动画
- 🎆 **粒子效果** - 特殊场景的粒子动画

### 4. 智能分割算法 (Smart Splitting)
**灵感来源**: 自然语言处理, 文档分页算法
- 🧠 **语义分割** - 按标题、段落智能分页
- 📏 **长度控制** - 可调节每页字符数(200-800)
- 🔗 **内容连贯** - 保持语义完整性
- 📄 **批量处理** - 一键生成多页卡片
- 🏷️ **页码标识** - 自动添加页面编号

### 5. 高级导出系统 (Advanced Export)
**灵感来源**: Figma导出, Canva导出功能
- 🖼️ **多格式支持** - PNG、JPG、PDF
- 🎯 **质量控制** - 0.5x到3x分辨率选择
- 📱 **平台预设** - Instagram、小红书等尺寸
- 🏷️ **水印功能** - 可添加个人品牌标识
- 📦 **批量导出** - 一键下载所有分页

```

├── index.html              # 主界面
├── style.css              # 样式系统
├── script.js              # 核心逻辑

├── animation-effects.js   # 动画效果模块
└── README.md             # 使用说明
```
