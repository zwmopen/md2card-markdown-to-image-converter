# MD2Card - Markdown转小红书卡片工具

一个本地化的Markdown转卡片工具，可以将Markdown文本转换为精美的小红书风格卡片图片。

## ✨ 功能特点

- 🎨 **多种主题色彩** - 5种精美渐变主题可选
- 📝 **实时预览** - 输入即时显示效果
- 💾 **一键导出** - 高清PNG图片下载
- 📱 **响应式设计** - 完美适配各种屏幕
- ⌨️ **快捷键支持** - 提高使用效率
- 🎛️ **样式自定义** - 字体大小、卡片宽度可调
- ✂️ **智能分割** - 长文本自动分页处理
- 📚 **批量下载** - 一键下载所有分页卡片

## 🚀 快速开始

### 方法一：直接使用
1. 下载所有文件到本地文件夹
2. 双击打开 `index.html` 文件
3. 开始使用！

### 方法二：本地服务器（推荐）
```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js
npx serve .

# 然后访问 http://localhost:8000
```

## 📖 使用说明

### 基本操作
1. **编辑内容**：在左侧编辑器输入Markdown文本
2. **实时预览**：右侧会实时显示卡片效果
3. **调整样式**：使用控制面板调整主题、字体大小等
4. **导出图片**：点击"下载图片"按钮保存为PNG

### 支持的Markdown语法
- **标题**：`# H1` `## H2` `### H3`
- **强调**：`**粗体**` `*斜体*`
- **列表**：`- 无序列表` `1. 有序列表`
- **引用**：`> 引用文本`
- **代码**：`` `行内代码` `` 和 ```代码块```
- **分割线**：`---`

### 自动分割功能
1. **启用分割**：勾选"自动分割长文本"选项
2. **调整参数**：设置每页最大字符数（200-800字符）
3. **智能分割**：系统按标题、句子智能分页
4. **分页浏览**：使用导航按钮查看各页内容
5. **批量下载**：一键下载所有分页卡片

### 分割算法特点
- **标题优先**：遇到新标题时优先分页
- **句子完整**：在句号、问号、感叹号处寻找分割点
- **长度控制**：确保每页内容不超过设定字符数
- **内容连贯**：保持段落和语义的完整性

### 快捷键
- `Ctrl/Cmd + S`：下载当前卡片
- `Ctrl/Cmd + K`：清空内容

## 💡 使用技巧

### 内容编写
1. **合理分段**：使用标题和空行合理分割内容
2. **标题层级**：善用H1-H6标题，便于自动分割
3. **列表整理**：使用列表让内容更有条理
4. **重点突出**：用**粗体**和*斜体*强调关键信息
5. **引用应用**：重要内容使用引用格式突出显示

### 分割优化
1. **字符控制**：建议每页300-500字符，确保视觉效果
2. **内容完整**：避免在句子中间分割
3. **逻辑清晰**：每页保持独立的主题或观点
4. **标题引导**：用标题作为分页的天然分割点

## 🐛 常见问题

**Q: 为什么图片下载失败？**
A: 请确保浏览器支持html2canvas，建议使用Chrome或Firefox最新版本。

**Q: 可以自定义更多样式吗？**
A: 可以修改CSS文件中的样式变量来自定义外观。

**Q: 支持哪些图片格式？**
A: 目前支持PNG格式导出，质量为最高清晰度。

**Q: 自动分割如何工作？**
A: 系统会根据标题、句子和字符数智能分割内容，确保每页内容完整且美观。

**Q: 可以批量下载所有卡片吗？**
A: 可以！启用自动分割后，点击"下载全部"按钮即可批量下载所有分页卡片。

**Q: 分割后的卡片顺序如何？**
A: 系统会自动为每张卡片添加页码（如：page1, page2），方便识别顺序。
